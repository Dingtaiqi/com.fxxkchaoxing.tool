# 学习通自动答题

> 📱 **LSPosed 模块** | 自动捕获学习通题目并调用 AI 智能答题，支持单选题、多选题、填空题自动作答。

## 功能特性

| 功能 | 说明 |
|------|------|
| **自动捕获** | 监听学习通 WebView 页面加载，自动获取题目 |
| **AI 答题** | 调用 API 智能分析题目并自动选择答案 |
| **自动填空** | 识别填空题并自动填充答案 |

## ⚠️ 仅供学习

本项目仅供学习交流使用，请勿用于任何商业目的或违规行为。

## 环境要求

- LSPosed 框架
- Android 9.0+ (API 29+)
